# sistema de entrenamiento cognitivo adaptativo

## objetivo
Diseñar un sistema de entrenamiento cognitivo adaptativo basado en modelos de dificultad dinámica y seguimiento del rendimiento del usuario.

## estructura del proyecto

00_REFERENCIAS/           # análisis y requisitos
01_PLAN_MAESTRO/          # visión y decisiones estratégicas
02_CASOS_DE_USO/          # requisitos funcionales
03_ARQUITECTURA/          # diseño arquitectónico (c4, hexagonal, adrs)
04_MODELO_DATOS/          # dominio y persistencia
05_AGENTES_IA/            # lógica de agentes inteligentes
06_INTERFAZ_USUARIO/      # diseño ui/ux
07_IMPLEMENTACION/        # guías técnicas y convenciones
08_CODIGO_FUENTE/         # código fuente principal
