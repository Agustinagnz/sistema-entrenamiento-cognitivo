# Requisitos Funcionales

## Introducción

Los requisitos funcionales describen las capacidades y servicios que debe proporcionar el Sistema de Entrenamiento Cognitivo Adaptativo.

---

## Gestión de Usuarios

### RF-01. Registro de usuarios

El sistema deberá permitir registrar usuarios y almacenar su información básica.

### RF-02. Consulta de usuarios

El sistema deberá permitir consultar la información de los usuarios registrados.

### RF-03. Modificación de usuarios

El sistema deberá permitir actualizar la información de un usuario.

---

## Sesiones de Entrenamiento

### RF-04. Inicio de sesión de entrenamiento

El sistema deberá permitir iniciar una sesión de entrenamiento asociada a un usuario.

### RF-05. Finalización de sesión

El sistema deberá permitir finalizar una sesión y almacenar las métricas obtenidas.

### RF-06. Gestión del estado de sesión

El sistema deberá mantener el estado de una sesión mientras permanezca activa.

---

## Ejercicios Cognitivos

### RF-07. Generación de ejercicios

El sistema deberá generar ejercicios de memoria, lógica y atención.

### RF-08. Parametrización de la dificultad

El sistema deberá ajustar los parámetros del ejercicio en función del nivel de dificultad requerido.

### RF-09. Presentación de ejercicios

El sistema deberá proporcionar al usuario una instancia concreta del ejercicio generado.

---

## Evaluación

### RF-10. Recepción de respuestas

El sistema deberá recibir las respuestas proporcionadas por el usuario.

### RF-11. Evaluación de respuestas

El sistema deberá determinar si una respuesta es correcta o incorrecta.

### RF-12. Medición del tiempo de respuesta

El sistema deberá registrar el tiempo empleado por el usuario para responder.

---

## Métricas y Adaptación

### RF-13. Generación de métricas

El sistema deberá calcular métricas de desempeño asociadas a cada respuesta.

### RF-14. Adaptación de la dificultad

El sistema deberá modificar el nivel de dificultad de acuerdo con el rendimiento reciente del usuario.

### RF-15. Actualización del perfil cognitivo

El sistema deberá actualizar el perfil cognitivo utilizando la información acumulada.

---

## Persistencia y Trazabilidad

### RF-16. Registro de eventos

El sistema deberá almacenar los eventos relevantes ocurridos durante una sesión.

### RF-17. Conservación del historial

El sistema deberá mantener el historial de sesiones del usuario.

### RF-18. Consulta del progreso histórico

El sistema deberá permitir visualizar la evolución del rendimiento del usuario.

---

## Administración

### RF-19. Configuración del sistema

El sistema deberá permitir administrar parámetros globales.

### RF-20. Gestión de ejercicios

El sistema deberá permitir administrar las plantillas de ejercicios disponibles.
