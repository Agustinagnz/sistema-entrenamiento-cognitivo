# Alcance del Sistema

## Alcance general

El Sistema de Entrenamiento Cognitivo Adaptativo tiene como finalidad proporcionar sesiones de entrenamiento cognitivo personalizadas mediante la adaptación dinámica de la dificultad de los ejercicios en función del desempeño del usuario.

La versión inicial del sistema se orienta a validar la arquitectura y las reglas de adaptación utilizando un conjunto reducido de funcionalidades esenciales.

## Funcionalidades incluidas

La primera versión del sistema incluye:

* Gestión básica de usuarios.
* Persistencia del perfil cognitivo.
* Ejecución de sesiones de entrenamiento.
* Ejercicios correspondientes a los dominios de memoria, lógica y atención.
* Evaluación automática de respuestas.
* Generación de métricas de desempeño.
* Adaptación dinámica del nivel de dificultad mediante reglas determinísticas.
* Registro de eventos para mantener trazabilidad del proceso.
* Consulta del progreso histórico del usuario.

## Funcionalidades excluidas

La versión inicial no contempla:

* Sistemas de interacción social entre usuarios.
* Chat o mensajería.
* Gamificación avanzada.
* Rankings globales.
* Generación de ejercicios mediante inteligencia artificial externa.
* Modelos de aprendizaje automático o deep learning.
* Sincronización en tiempo real entre múltiples dispositivos.
* Dashboards analíticos avanzados.

## Funcionalidades previstas para futuras versiones

Podrán incorporarse en etapas posteriores:

* Gráficos de evolución del rendimiento.
* Exportación de información en PDF o CSV.
* Simulación automática de sesiones para pruebas.
* Modo de entrenamiento libre sin adaptación.
* Ranking entre usuarios.
* Integración con servicios externos para generación de ejercicios.

## Restricciones del proyecto

El sistema será desarrollado con fines educativos, priorizando:

* Simplicidad de implementación.
* Separación clara de responsabilidades.
* Mantenibilidad.
* Escalabilidad futura.
* Uso de tecnologías de código abierto.

La complejidad se mantendrá controlada para favorecer el aprendizaje de arquitectura de software y buenas prácticas de desarrollo.
